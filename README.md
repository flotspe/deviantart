# Sync a chosen DeviantArt gallery to your top N most-favourited deviations.

status: **Proof of Concept**

required packages: python-dotenv, requests

## usage

step 0. have a gallery called Top 20 Favorites on DeviantArt  
step 1. register your app (https://github.com/jmelloy/dastats/blob/main/README.md but I use the URI http://127.0.0.1:8123/)  
step 2. rename .env.sample to .env and fill in client id and client secret  
step 3. run main.py
